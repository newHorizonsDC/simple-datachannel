simple-datachannel
==================

A simple WebRTC datachannel establishment demo.
The explanation you can find in [my blog post][blog].

## Prerequisites

- Node and NPM

## How to run

In the terminal type the following:

```
npm install
node ws-server.js
```

Now go to [http://localhost:8888][link]
